# IPL Data Insights & Visualization using Power BI

## 📊 Project Overview
This project analyzes **18 seasons of IPL data (2008–2025)** using Power BI.  
It transforms raw datasets (matches, ball-by-ball, players, teams) into interactive dashboards that reveal batting, bowling, toss, venue, and season trends.

## ⚡ Key Features
- Batting & Bowling Analysis
- Season-on-Season Trends
- Player Comparison Dashboard

## 📂 Files
- `IPL_performance_analysis.pbix` → Power BI dashboard
- `dashboard.reports.docx` → Detailed project report
- `datasets/` → CSV files (matches, ball-by-ball, players, teams)

## 🚀 How to Use
1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop).
2. Open `IPL_Dashboard.pbix`.
3. Explore interactive dashboards with filters, drill-through, and bookmarks.
